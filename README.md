# Klasifikasi Felidae Menggunakan TFLite Web

Aplikasi Klasifikasi Felidae adalah aplikasi berbasis web yang dibangun dengan Flutter, dirancang untuk mengklasifikasikan spesies kucing besar (Felidae) dari gambar yang diunggah menggunakan model *machine learning* TensorFlow Lite (TFLite).

## Navigasi Aplikasi

Aplikasi ini memiliki empat navigasi utama untuk mempermudah pengalaman pengguna:

1.  **Home (Beranda)**
2.  **Input (Klasifikasi)**
3.  **History (Riwayat)**
4.  **Profile (Profil)**

---

## 1. Home (Beranda)

Halaman ini berfungsi sebagai direktori informasi. Pengguna dapat melihat daftar lengkap semua kelas spesies Felidae yang dapat diidentifikasi oleh model klasifikasi:

* **Cheetah**
* **Leopard**
* **Lion**
* **Puma**
* **Tiger**

Pengguna dapat menekan salah satu nama spesies dari daftar tersebut untuk melihat informasi detail, yang meliputi **Nama Spesies** dan **Deskripsi** singkat mengenai karakteristik spesies tersebut.

## 2. Input (Klasifikasi)

Ini adalah fitur utama aplikasi untuk menguji model klasifikasi.

### Cara Menggunakan Klasifikasi

1.  **Mengunggah Gambar:** Pengguna mengambil gambar dari **penyimpanan** lokal perangkat (galeri/file) dan mengunggahnya ke antarmuka aplikasi.
2.  **Proses Klasifikasi:** Setelah gambar dimuat, pengguna menekan tombol **Classification**.
3.  **Melihat Hasil (Result):** Aplikasi akan memproses gambar menggunakan model TFLite dan menampilkan **Nama Spesies** sebagai hasil klasifikasi (misalnya, *Tiger*).

## 3. History (Riwayat)

Halaman **History** berfungsi sebagai log aktivitas. Pengguna dapat melihat dan meninjau kembali semua hasil klasifikasi yang telah dilakukan sebelumnya. Setiap entri riwayat mencakup gambar yang diunggah dan hasil klasifikasi yang diperoleh.

## 4. Profile (Profil)

Halaman **Profile** memungkinkan pengguna untuk mengelola informasi akun mereka, khususnya:

* **Edit Username:** Mengubah nama pengguna (username) yang digunakan dalam aplikasi.
* **Edit Foto Profil:** Mengganti foto profil pengguna. Opsi untuk mengganti foto profil dapat diambil dari:
    * **Penyimpanan** perangkat lokal.
    * **Carousel Slider** yang menyediakan pilihan foto profil default.
